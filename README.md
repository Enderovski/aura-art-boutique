# 🎨 Aura Art Boutique

A modern, highly-responsive web application built to showcase beautiful UI/UX designs and fast frontend architecture.

## 🚀 Built With
- **React 18**
- **Vite**
- **Tailwind CSS**
- **TypeScript**

## 💡 About The Project
This project was scaffolded using AI-assisted tools (Lovable/Vite) to rapidly test frontend implementations, dynamic routing, and component-driven architecture. It serves as a playground for modern web features and optimized performance.
